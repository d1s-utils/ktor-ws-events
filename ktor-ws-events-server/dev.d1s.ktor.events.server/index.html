val WebSocketEvents: ApplicationPlugin<WebSocketEventsConfiguration>

Enables support for event streaming over WebSockets. Supposed to be used in pair with webSocketEvents route builder. Once the plugin is installed, it will fire a job listening for events in the provided WebSocketEventsConfiguration.channel.
